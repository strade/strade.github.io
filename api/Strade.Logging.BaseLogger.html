    Log(LogSeverity, Object, Exception, String, Object[])

    Logs a message with the specified severity from the specified sender with the specified message, args and exception.

    Declaration
    protected abstract void Log(LogSeverity severity, object sender, Exception exception, string message, params object[] args)
    Parameters
    Type Name Description
    LogSeverity severity

    the log severity.
    Object sender

    the message sender.
    Exception exception

    the exception that occurred; or null if not present.
    String message

    the log message.
    Object[] args

    the message format arguments; if null message formatting is not applied to the specified message.
    Exceptions
    Type Condition
    ArgumentNullException

    thrown if the specified sender is null.
