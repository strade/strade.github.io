    ReferenceType

    Gets the type of the reference for the cache.

    Declaration
    CacheReferenceType ReferenceType { get; }
    Property Value
    Type Description
    CacheReferenceType

    the type of the reference for the cache.

    NoneDenotes that the entry should not be cached.
    Strong Denotes that the entry holds a strong reference to the cached value.
    Tracking Denotes that the entry is hold until the guaranteed lifetime expires, after this the cache entry will become a weak reference to the strong value handle.
    Weak Denotes that the entry holds a weak reference to the cached value.
