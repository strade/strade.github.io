    TotalLength

    Gets the total length of all fields together.

    Declaration
    public int TotalLength { get; }
    Property Value
    Type Description
    Int32

    the total length of all fields together.
    Remarks

    The length includes the lengths of.

    • Embed Title ( Title)
    • Embed Description ( Description)
    • All field names ( Name / Name)
    • All field names ( Value / Value)
    • Footer Text ( Text / Text)
    • Author Name ( Name / Name)

    The total length of all fields should be less than MaximumTotalLength characters.
